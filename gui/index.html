how to read the metrics

Compliance

A slot is compliant if all KPIs for a service are met. Rolling compliance is the ratio of compliant slots in the window. Global compliance = min(ETCS, Voice, Video).

Latency model

Edge latency is estimated from a rolling baseline (low‑percentile + smoothing + floor), then reduced by the edge factor and increased by compute cost. Partial SAT coverage blends raw and edge latency and adds a small detour penalty.
